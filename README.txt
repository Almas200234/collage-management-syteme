1. PROJECT DESCRIPTION:
This is a desktop-based application developed in Java using 
Swing and MySQL. It allows administrators to manage student 
records, including their name, department, enrollment, and contact.

2. PREREQUISITES:
- Java Runtime Environment (JRE) installed.
- MySQL Server (XAMPP recommended).
- MySQL Connector JAR (Already included in the project JAR).

3. DATABASE SETUP:
- Open XAMPP and Start Apache and MySQL.
- Go to http://localhost/phpmyadmin/
- Create a new database named: college_db
- Select 'college_db' and click on the 'Import' tab.
- Choose the 'college_db.sql' file provided in this folder.
- Click 'Go' to create the tables.

4. HOW TO RUN THE APPLICATION:
- Locate the 'CollegeManagement.jar' file.
- Double-click the file to launch.
- If it doesn't open, right-click and select 'Open with' -> 'Java(TM) Platform SE binary'.

5. LOGIN CREDENTIALS:
- Username: Admin
- Password: password
